# Attention U-Net for LIDC (Full Project Bundle)

This package contains:
- Full source code (PyTorch)
- Sample images for smoke testing
- Draft paper (markdown)
- Case study
- Architecture placeholder image

## Quickstart
1. Create environment:
   ```
   pip install -r code/requirements.txt
   ```
2. Train model:
   ```
   python code/src/train.py --config code/configs/exp1.yaml
   ```

Replace sample data paths with actual LIDC paths when training on real data.
